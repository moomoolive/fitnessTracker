# Fitty Client

## Dev Tools
* Typescript
* ESlint
* Jest
* SASS

### Run development server with hot reload
```
npm run dev
```

### Build production bundle
```
npm run build
```

### Run production server
```
npm run start
```

### Run linting
```
npm run lint
```

### Run diagnostics on test library
```
npm run test:debug
```

### Run unit tests
```
npm run test:unit
```
